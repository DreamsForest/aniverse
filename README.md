# AniVerse — каталог аниме

Красивый тёмный сайт-каталог аниме (neon / glassmorphism). Без бэкенда: чистый
HTML/CSS/JS + файл-каталог `data.js`.

Видео отдаёт **Kodik**: сайт по `shikimoriID` тайтла на лету запрашивает у Kodik
ссылку на плеер и встраивает его в iframe. Список серий, сезоны и озвучки
показываются и переключаются прямо внутри плеера и обновляются автоматически —
вручную серии вести не нужно. Своего видео мы не храним → хостинг бесплатный.

## Структура

```
anime-site/
├── index.html            — каркас SPA (главная/каталог/тайтл/плеер)
├── styles.css            — весь дизайн
├── app.js                — логика: роутинг, поиск, фильтры, сортировка, история, плеер
├── data.js               — КАТАЛОГ (автогенерируется)
├── anime/<id>/index.html — SEO-страницы по тайтлам (автогенерируются)
├── sitemap.xml           — карта сайта (автогенерируется)
├── robots.txt, favicon.svg, site.webmanifest
├── tools/gen-data.mjs    — собрать каталог топ-популярных с Shikimori + проверка Kodik
├── tools/add-season.mjs  — добавить новинки сезона/года (по умолчанию 2026)
├── tools/gen-seo.mjs     — собрать SEO-страницы + sitemap + проставить домен
└── README.md
```

## Обновить / расширить каталог

Каталог `data.js` собран автоматически: топ популярных аниме с Shikimori
(русские названия, жанры, описания, постеры), причём каждый тайтл проверен на
наличие в Kodik — поэтому «Смотреть» работает у всех карточек.

Чтобы пересобрать или увеличить каталог:

```bash
node tools/gen-data.mjs                       # топ-популярных (по умолчанию 120)
TARGET=200 PAGES=8 node tools/gen-data.mjs    # больше тайтлов
node tools/add-season.mjs                     # добавить новинки 2026 (не трогая остальное)
SEASON=2025 node tools/add-season.mjs         # добавить другой год
```

> Требуется Node.js 18+. `gen-data.mjs` перезаписывает `data.js` целиком,
> `add-season.mjs` только дописывает новые тайтлы.
>
> ⚠ После любого изменения каталога перегенерируй SEO-страницы:
> `node tools/gen-seo.mjs` (см. раздел SEO).

## Как запустить локально

Просто открой `index.html` двойным кликом — сайт заработает в браузере.
(Опционально для «чистого» окружения можно поднять локальный сервер:
`python -m http.server` и зайти на http://localhost:8000)

## Как добавить аниме вручную

Обычно проще пересобрать каталог генератором (см. выше). Но можно дописать тайтл
руками — достаточно знать его **Shikimori ID**. Открой аниме на
https://shikimori.one — число в адресе и есть ID.
Пример: `shikimori.one/animes/40748-jujutsu-kaisen` → `40748`.

Открой `data.js` и добавь объект в массив `ANIME`:

```js
{
  id: "naruto",                  // уникальный код латиницей (для ссылок на сайте)
  shikimoriID: 20,               // ГЛАВНОЕ: ID с shikimori.one — по нему грузится плеер
  title: "Наруто",
  original: "Naruto",
  year: 2002,
  genres: ["Экшен", "Сёнен"],
  rating: 8.3,
  status: "Завершён",            // Онгоинг | Завершён | Анонс
  poster: "https://.../naruto.jpg", // можно оставить "" — будет красивая заглушка
  description: "Описание..."
}
```

Серии, сезоны и озвучки подтянутся из Kodik автоматически.

### Про токен Kodik

В `data.js` есть `KODIK_TOKEN` — публичный токен плеера Kodik (тот же, что в их
официальном виджете). Если плеер вдруг перестанет находить аниме, обнови токен:
открой https://kodik-add.com/add-players.min.js и скопируй значение `token="..."`.

## Бесплатный хостинг (рекомендованный путь)

Поскольку это статический сайт, подойдёт любой из вариантов. Самый простой:

### Вариант 1 — Netlify (drag & drop, без аккаунта Git)
1. Зайди на https://app.netlify.com/drop
2. Перетащи папку `anime-site` в окно браузера.
3. Готово — получишь ссылку вида `https://твой-сайт.netlify.app`.

### Вариант 2 — Cloudflare Pages (быстрый CDN, свой домен бесплатно)
1. Создай репозиторий на GitHub и залей туда файлы.
2. https://pages.cloudflare.com → Create project → подключи репозиторий.
3. Build command: пусто, Output dir: `/` (корень). Deploy.

### Вариант 3 — GitHub Pages
1. Создай репозиторий, залей файлы.
2. Settings → Pages → Source: `main` / `(root)` → Save.
3. Сайт появится на `https://username.github.io/repo/`.

Все три варианта — бесплатно и без лимитов на видео (видео ведь не у нас,
а на внешних плеерах).

## SEO — как попасть в топ поисковиков

### Что уже сделано в коде
- **Отдельная страница на каждый тайтл** (`/anime/<id>/`) с уникальными `<title>`,
  `description`, Open Graph и микроразметкой Schema.org (`TVSeries`). Это главное:
  поисковики не видят то, что после `#`, поэтому SPA-страницы (`#/title/...`) сами
  по себе не ранжируются, а статические `/anime/<id>/` — ранжируются.
- `sitemap.xml`, `robots.txt`, `canonical`, мета-теги и заголовок на главной.
- `favicon.svg`, `site.webmanifest`, быстрый статический сайт (хорошо для ранжирования).
- Внутренняя перелинковка («Похожие аниме» на каждой странице).

### Шаги, которые нужно сделать руками (по важности)

1. **Свой домен.** Поддомены вида `*.netlify.app` ранжируются хуже. Купи короткий
   домен (.ru/.com). Затем перегенерируй страницы с этим доменом — он проставится
   везде (canonical, og, sitemap, robots, index.html):
   ```bash
   SITE_URL=https://твойдомен.ру node tools/gen-seo.mjs
   ```
   Запускай эту команду после каждого изменения каталога.

2. **Search Console и Яндекс.Вебмастер.** Для русскоязычной аудитории Яндекс не
   менее важен, чем Google.
   - Google: https://search.google.com/search-console → добавь сайт → отправь `sitemap.xml`.
   - Яндекс: https://webmaster.yandex.ru → добавь сайт → отправь `sitemap.xml`.

3. **HTTPS и скорость.** Netlify/Cloudflare дают HTTPS автоматически. Cloudflare
   Pages — самый быстрый CDN (скорость = фактор ранжирования).

4. **Контент = трафик.** Поисковый трафик аниме-сайтов идёт по запросам
   «смотреть <название> онлайн». Чем больше тайтлов в каталоге (`gen-data.mjs`,
   `add-season.mjs`) — тем больше страниц и запросов ты охватываешь.
   Добавляй уникальные описания (не копии с других сайтов).

5. **Ссылки на тебя (backlinks).** Упоминания в тематических пабликах, на
   форумах, в подборках. Чем больше качественных ссылок — тем выше позиции.

6. **og-картинка.** Положи в корень `og-cover.jpg` (1200×630) — красивое превью
   при репостах ссылки повышает кликабельность.

### Чего ожидать
SEO — небыстрая история: индексация занимает дни-недели, рост позиций — месяцы.
Решают: уникальный контент, объём каталога, скорость, домен и ссылки.

> Ниша «пиратского» аниме высококонкурентна и юридически рискованна (см. ниже) —
> это влияет и на возможность продвижения в легальных рекламных каналах.

## Важно про контент

Размещайте только то аниме, на которое у вас есть права/лицензия, либо
встраивайте легальные источники. Ответственность за контент — на владельце сайта.
